# 隐私与安全方案

## 1. 隐私定位

Taskora 必须从产品表达、默认设置和技术实现上都远离“监控软件”。

推荐定位：

- 个人工作记忆。
- 本地优先复盘工具。
- 用户可控的任务时间线。

不推荐定位：

- 员工监控。
- 屏幕监察。
- 行为审计。
- 效率打分系统。

## 2. 默认记录范围

默认记录：

- 前台应用进程名。
- 应用显示名。
- 窗口标题。
- 应用使用时间段。
- 切换次数。
- 用户空闲时间。
- 任务开始、暂停、完成时间。
- 用户手动进展。

默认不记录：

- 键盘逐字输入。
- 密码。
- 聊天内容。
- 剪贴板内容。
- 截图。
- 录屏。
- OCR 屏幕文字。
- 浏览器页面正文。

## 3. 隐私规则

隐私软件黑名单：

- 微信。
- QQ。
- Telegram。
- 密码管理器。
- 银行客户端。
- 支付软件。
- 浏览器隐私模式。

规则类型：

- process_name exact。
- process_name contains。
- window_title contains。
- window_title regex。
- website_domain，后续浏览器插件使用。

动作：

- exclude：完全不记录。
- redact_title：记录应用，但隐藏窗口标题。
- mark_private：记录时间段为隐私时间，不保存原始标题。

MVP 默认建议：

- 聊天软件 mark_private。
- 密码管理器 exclude。
- 银行/支付 redact_title 或 exclude。
- 浏览器隐私模式 redact_title。

## 4. AI 隐私边界

AI 调用前必须展示：

- 将发送哪些任务。
- 将发送哪些进展。
- 是否包含应用统计。
- 是否包含窗口标题。
- 时间范围。
- provider 和 endpoint。

默认规则：

- private 数据不发送。
- 隐藏标题不发送。
- 用户删除的数据不发送。
- API key 由用户配置或本地模型提供。

AI 总结保存时：

- 保存输出。
- 保存 input hash。
- 可选保存 input snapshot。
- 用户可删除 AI 输出和 input snapshot。

## 5. 用户控制

必须提供：

- 一键暂停记录。
- 一键恢复记录。
- 删除某一天记录。
- 删除某个任务记录。
- 删除所有活动记录但保留任务。
- 导出数据库或 Markdown。
- 隐私软件黑名单编辑。
- AI 前确认数据范围。

UI 要求：

- 托盘菜单中显示“记录中/已暂停”。
- Settings 一级页显示 Privacy。
- Today 页面显示当前记录状态。
- 命中 privacy 时在 UI 中清楚显示“已隐藏”。

## 6. 数据安全

本地文件：

- `taskora.db` 存储结构化数据。
- `taskora.settings.json` 存储非敏感配置。
- API key 使用 Windows Credential Manager 或 DPAPI。
- Markdown 导出目录由用户选择。

建议：

- 数据库默认不加密，先降低 MVP 复杂度。
- 提供“导出/备份/删除”能力。
- 后续提供 SQLCipher 或系统级加密选项。

## 7. 避免被认为是监控软件

产品层：

- 首次启动明确说明默认记录范围。
- 不用“监控”“审计”“捕获”等词。
- 不做团队后台。
- 不提供管理员查看他人记录能力。
- 不展示“效率分数”作为核心卖点。

功能层：

- 不默认截图、录屏、键盘记录。
- 不读取聊天内容。
- 不自动上传云端。
- AI 前必须确认范围。
- 用户能删除任意记录。

技术层：

- 不使用全局键盘 hook。
- 不使用屏幕采集 API。
- 不绕过隐私软件。
- 日志中不写原始敏感标题。

## 8. 威胁模型

主要风险：

- 用户误以为 Taskora 记录了聊天或密码。
- 窗口标题中包含敏感客户名、文档名、网页标题。
- AI 请求把敏感任务发送到云模型。
- 本地数据库被其他人读取。
- 团队版被用于员工监控。

缓解：

- 隐私 onboarding。
- 黑名单默认模板。
- title redaction。
- AI confirm dialog。
- 本地导出和删除。
- 后续数据库加密。
- 团队版单独合规评估，不从个人版直接扩展。

## 9. 隐私文案示例

首次启动：

```text
Taskora 默认只记录你正在使用的应用、窗口标题和任务时间，用于生成个人复盘。
它不会记录键盘输入、聊天内容、剪贴板、截图或录屏。
所有数据默认保存在你的电脑上。发送给 AI 前，你会先看到并确认数据范围。
```

AI 确认：

```text
以下内容将用于生成今日总结。隐私应用和已隐藏窗口标题不会发送。
你可以取消勾选任何任务、应用或时间段。
```

暂停记录：

```text
工作记录已暂停。Taskora 不会继续记录前台应用和窗口标题，直到你手动恢复。
```
